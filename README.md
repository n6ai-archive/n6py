N6 AI Python Tools.

## About

Tooling for common problems in Scientific Computing, Machine Learning and Deep Learning.

## Installing

**pip**
```sh
pip install n6py
```

**Poetry**
```sh
poerty add n6py
```

## Using

To use the `n6py` package import it into your project.

```py
import n6py as n6
```
